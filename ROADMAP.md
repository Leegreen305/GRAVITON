# GRAVITON Roadmap

## v0.1.0 (Current)

- [x] Alcubierre warp drive metric, energy density, bubble dynamics
- [x] Gravitoelectromagnetic field equations, Lense-Thirring frame dragging
- [x] Casimir effect, quantum vacuum energy, exotic matter constraints
- [x] Geodesic solver for Schwarzschild and Kerr spacetimes
- [x] Interactive Plotly 3D visualizations
- [x] Rich terminal dashboard
- [x] Full pytest test suite (56 tests)
- [x] GitHub Actions CI

## v0.2.0 (Planned)

- [ ] Symbolic tensor computation with SymPy (Riemann, Ricci, Einstein tensors)
- [ ] Morris-Thorne traversable wormhole metric
- [ ] Krasnikov tube spacetime
- [ ] Natario warp drive variant
- [ ] VPython real-time 3D warp bubble animation
- [ ] Parameter optimization: minimize exotic energy for given warp velocity

## v0.3.0 (Planned)

- [ ] Van den Broeck modified warp metric (reduced energy requirements)
- [ ] White-Juday warp field interferometer simulation
- [ ] Gravitational wave emission from warp bubbles

- [ ] Parallel computation with multiprocessing for large grids
